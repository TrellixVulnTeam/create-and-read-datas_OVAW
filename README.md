# Application for create and read datas using: Flask, MongoDb, AngularJs e Material Design Lite

## Dependences

1. Install [Python 3.7.0](https://www.python.org/downloads/3.7.0)

Details: 
- Default installing, click in Add Path
- Access the CMD and type: python to verify that the configuration is right
- If you have problems setting up [access here](https://python.org.br/instalacao-windows/) 
for config in Windows
- Exit CMD

2. Install [MongoDB 4.0](https://www.mongodb.com/download-center#community)

Details: 
- Default installing

## Building an environment

Access the CMD and enter the directory you want to place the project

1. In prompt type it for create environment isolated: 

pip install virtualenv

2. After installing virtualeng, type it for create the project folder:

virtualenv projectname

3. Type it for enter in project folder:

cd projectname

4. Paste the files of [this repository]() here (projectname folder)

5. Enter in folder scripts, type it in CMD:

cd scripts

6. Type it for activate the environment:

activate

7. Return to main folder, type it:

cd..

8. Install flask, type it:

pip install flask

9. Install PyMongo, type it:

pip install pymongo

10. Execute the project, type it:

python app.py

Show the message that project is running in 127.0.0.1:5000

Access the address 127.0.0.1:5000 for browser
